# Hemant Pachauri — Full-Stack & AI/Security Engineer Portfolio

A modern, terminal-themed developer portfolio built to showcase projects across full-stack development, applied AI/ML, and network security.

**Live Preview**: [https://hemantportfolio-theta.vercel.app/](https://hemantportfolio-theta.vercel.app/)

## ✨ Features

- **Retro Terminal Aesthetic**: Clean, developer-focused UI featuring authentic monospace typography and custom pixel fonts (`Pixelbasel`).
- **Interactive Matrix Background**: A custom `Three.js` implementation rendering a responsive, interactive 3D dotted matrix that sits behind the application.
- **Glassmorphism UI**: Premium translucent cards with `backdrop-blur` that allow the animated background to subtly shine through the components.
- **Scroll Animations**: Smooth, hardware-accelerated intersection observer animations that reveal content as the user scrolls.

## 🛠️ Tech Stack

- **Frontend Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Rendering**: [Three.js](https://threejs.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: Configured for seamless deployment on platforms like Vercel or Netlify.

## 🚀 Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hemant4true/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
